Basisprincipes Informatica

1. Variabelen

In Python gebruik je een variabele om informatie te onthouden. Een variabele heeft een naam en een waarde. Je assigneert (geeft) een waarde aan een naam met het = teken. Denk aan een variabele als een doosje met een label erop waar je iets in stopt.

# Toewijzing (assignatie) van een variabele a met waarde 2
a = 2

# Toewijzing (assignatie) van een variabele b met waarde 3
b = 3

# Toon de waarde in een variabele
print(a)  # Op het scherm verschijnt: 2

Tip: Gebruik steeds duidelijke, korte namen voor variabelen.

2. Datatypes

Elke waarde in Python heeft een type. Dit bepaalt wat je ermee kunt doen (bijv. rekenen met getallen of plakken met tekst). Denk aan een doos voor koeken en een doos voor boeken: allebei dozen maar met een verschillende functie.

# Integer (int): Gehele getallen
score = 10

# Float: Kommagetallen (gebruik een punt!)
prijs = 19.95

# String (str): Tekst, altijd tussen " " of ' '
naam = "Angela"

# Boolean (bool): Waarheidsinstelling
is_gelukt = True
is_niet_gelukt = False

Tip: Een variabele zal automatisch herkennen welk type dit is. Verschillende types kan je niet zomaar combineren. Een variabele omzetten van het ene naar het andere type heet typecasting en doe je met int(variabele), str(variabele), ...

3. Vergelijken

Met vergelijkingsoperatoren kan je controleren of waarden gelijk zijn of niet. Vergelijkingen of condities geven altijd een boolean terug.

a == b betekent: Is a gelijk aan b?

a != b betekent: Is a niet gelijk aan b?

x = 5

print(x == 5)  
# True

print(x == 3)  
# False 

print(x != 5)  
# False 

print(x != 3)  
# True 

print(x == '5')  
# False

4. Lists

Een lijst gebruik je om meerdere variabelen in 1 datastructuur op te slaan. De volgorde van de variabelen is van belang. Je kan een element opvragen door de lijst te indexeren op een positie. De lijst begint steeds te tellen vanaf positie 0.

# Een lijst met drie strings:
kleuren = ["rood", "blauw", "groen"]

# Het tweede element opvragen:
kleuren[0] # Op het scherm verschijnt: rood

# Een lijst met opeenvolgende integers aanmaken
print(range(2, 6)) # Op het scherm verschijnt: [2, 3, 4, 5]

5. Dictionaries

Een dictionary gebruik je om meerdere variabelen in 1 datastructuur op te slaan. Elk element in de dictionary bestaat nu uit een sleutel (key) en een waarde (value). Het type van de waarden en de volgorde van de sleutel-waardeparen maken niet uit.

auto = {"merk": "Tesla", "jaar": 2023}

# Nieuwe sleutel/waarde toevoegen:
auto["kleur"] = "rood"
print(auto) # Output: {"merk": Tesla, "jaar": 2023, "kleur": "rood"}

# Waarde opvragen via de sleutel:
print(auto["merk"]) # Output: Tesla

6. De lus

For-loop met range(): gebruik je wanneer je iets een bepaald aantal keer wil herhalen. Python maakt automatisch een lijst van opeenvolgende getallen.

for i in range(0, 3):
    print("Herhaling", i)

# range(0,3) maakt de getallen: 0, 1, 2
# Output:
# Herhaling 0
# Herhaling 1
# Herhaling 2

For-loop met een lijst: gebruik je wanneer je door bestaande data wil gaan.

namen = ["Alice", "Bob", "Eve"]

for naam in namen:
    print("Hallo", naam)

# De nieuwe gedefinieerde variabele 'naam' wordt telkens een element uit de lijst
# Output:
# Hallo Alice
# Hallo Bob
# Hallo Eve

While-loop: blijft herhalen zolang een voorwaarde True is.

energie = 5

while energie > 0:
    print("Ik werk nog...")
    energie -= 1

# De lus stopt wanneer energie 0 wordt

7. Functies

Wanneer je een stuk code wil groeperen, kan je een functie gebruiken. Een functie heeft soms een input die aan het begin wordt meegegeven. De code in een functie kan je oproepen wanneer of hoe vaak je wil. Pas wanneer je de functie oproept bepaal je de inputwaarden.

Hieronder zie je twee functies. return geeft een waarde terug zodat je die verder kan gebruiken, maar toont deze niet op het scherm. print() geeft geen waarde terug die je kan herbruiken, maar toont ze meteen op het scherm.

def bereken_btw(bedrag):
    totaal = bedrag * 1.21
    return totaal

factuur = bereken_btw(100)
print(factuur)

# Variabele 'factuur' bevat nu de waarde 121.0
# Op het scherm verschijnt niets.

def toon_btw(bedrag):
    totaal = bedrag * 1.21
    print(totaal)

resultaat = toon_btw(100)


# Op het scherm verschijnt: 121
# Variabele 'resultaat' is leeg want de functie gaf niets terug.

8. Recursie

Wanneer een functie zichzelf opnieuw aanroept noemen we dit recursie.

Er moet altijd een stopconditie (base case) zijn. Anders roept hij zichzelf een onbepaald aantal keer op.

def som_tot_nul(n):
    if n == 0:
        return 0
    else:
        return n + som_tot_nul(n - 1)
som_tot_nul(3)

# Wat gebeurt er bij som_tot_nul(3)?

som_tot_nul(3)
= 3 + som_tot_nul(2)
= 3 + (2 + som_tot_nul(1))
= 3 + (2 + (1 + som_tot_nul(0)))
= 3 + (2 + (1 + 0))
= 6
